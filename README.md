This is a web app to allow you to record the books that you have read. It allows adding books by barcode scanning, ISBN, or manual input. Mark books as read, reading or to read. Sort bookshelf by date or author. View stats.
The app works better if you obtain your own google books api-key - otherwise defaults to the public key which is limited.
The barcode scanner works best if you run in chrome. Chrome also allows you to save the page to your phone as a full page app (three dot menu: Add to home screen).
